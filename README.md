# SkillSwap

A peer-to-peer skill exchange platform where you can teach what you know and learn what you don't — no money required.

## About

SkillSwap connects people who want to trade expertise. Whether you're a guitarist who wants to learn graphic design, or a chef who wants to pick up coding basics, SkillSwap helps you find someone to swap with.

## Features

- **Browse Skills** — Explore skill listings across a range of categories
- **Create a Profile** — Showcase the skills you offer and the ones you're looking to learn
- **Authentication** — Sign up and log in with email/password or Google
- **Responsive Design** — Fully mobile-friendly UI

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, shadcn/ui
- **Backend & Auth:** Supabase
- **Testing:** Vitest, Playwright

## Getting Started

```bash
# Install dependencies
bun install

# Start the development server
bun run dev
```

## Running Tests

```bash
# Unit tests
bun run test

# E2E tests
bunx playwright test
```
